Incorporating Volatility, Multi-Period Returns, and Volume Adjustment for Robustness
├── Calculate Daily Log Returns
│   ├── Use Close price of day t
│   └── Use Close price of day t-1
│       └── Compute Log Return: log(Close_t / Close_{t-1})
├── Calculate 2-Period and 5-Period Log Returns
│   ├── 2-Period Log Return
│   │   ├── Use Daily Log Returns
│   │   └── Sum the last 2 days' Log Returns
│   └── 5-Period Log Return
│       ├── Use Daily Log Returns
│       └── Sum the last 5 days' Log Returns
├── Calculate Realized Volatility
│   ├── Use Daily Log Returns
│   └── Compute Standard Deviation over a Lookback Period (e.g., 20 days)
├── Adjust Returns by Realized Volatility
│   ├── Use 2-Period Log Return
│   ├── Use 5-Period Log Return
│   └── Divide each by Realized Volatility
├── Calculate Average Volume
│   ├── Use Volume of day t
│   ├── Use Volume of day t-1
│   └── Use Volume of day t-2
│       └── Compute Average Volume: (Volume_t + Volume_{t-1} + Volume_{t-2}) / 3
└── Final Alpha Factor: Adjusted Returns Weighted by Volume
    ├── Use 2-Period Log Return Adjusted by Volatility
    ├── Use 5-Period Log Return Adjusted by Volatility
    └── Multiply each by Average Volume
        └── Final Alpha Factor: (2-Period Log Return / Realized Volatility) * Average Volume
                                + (5-Period Log Return / Realized Volatility) * Average Volume
