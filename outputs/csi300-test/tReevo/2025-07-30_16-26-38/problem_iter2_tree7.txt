Price and Volume Momentum Alpha
├── Calculate Daily Close-to-Close Return
│   ├── Use Close price of day t
│   └── Use Close price of day t-1
│       └── Subtract t-1 close from t close
├── Calculate Accumulated Volume Trend (AVT)
│   ├── Initialize AVT to 0
│   ├── For each of the past N days
│   │   ├── If today's volume is greater than yesterday's volume
│   │   │   └── Add difference to AVT
│   │   └── Else
│   │       └── Subtract difference from AVT
├── Calculate Weighted Close-to-Close Return
│   ├── If AVT > 0
│   │   └── Multiply Close-to-Close Return by (1 + |AVT|)
│   └── If AVT < 0
│       └── Multiply Close-to-Close Return by (1 - |AVT|)
├── Incorporate Price Momentum
│   ├── Calculate Average Close-to-Close Return over the past M days
│   └── Add Weighted Close-to-Close Return to Average Close-to-Close Return
└── Determine Sign of AVT
    ├── If AVT > 0
    │   └── Positive trend, more weight on momentum
    └── If AVT < 0
        └── Negative trend, less weight on momentum
