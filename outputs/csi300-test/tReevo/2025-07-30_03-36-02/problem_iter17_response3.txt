```python
import pandas as pd
import numpy as np

def heuristics_v2(df):
    # Calculate Intraday Volatility
    intraday_volatility = df['high'] - df['low']
    
    # Weight by Volume
    weighted_volatility = intraday_volatility * df['volume']
    
    # Calculate Intraday Momentum
    daily_momentum = df['close'] - df['open']
    rolling_window = 5  # Define the rolling window size
    rolling_momentum = daily_momentum.rolling(window=rolling_window).sum()
    
    # Integrate Momentum and Volatility
    integrated_factor = weighted_volatility + rolling_momentum
    
    # Apply Dynamic Exponential Smoothing
    alpha = 0.9  # Define the smoothing factor
    smoothed_factor = integrated_factor.ewm(alpha=alpha, adjust=False).mean()
    
    # Ensure Values are Positive
    small_constant = 1e-6  # Small constant to avoid log of zero or negative
    smoothed_factor_positive = smoothed_factor + small_constant
    
    # Apply Logarithmic Transformation
    transformed_factor = np.log(smoothed_factor_positive)
    
    return transformed_factor
```

This function `heuristics_v2` calculates an alpha factor by integrating intraday volatility and momentum, applying dynamic exponential smoothing, and then transforming the result with a logarithm. The output is a pandas Series indexed by date, representing the factor values.
