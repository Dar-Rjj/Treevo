```txt
Compute Daily Range
├── Use High price
└── Use Low price
    └── Subtract low from high to get range

Calculate Intraday Return
├── Use High price
└── Use Low price
    ├── Subtract Low from High
    └── Divide by Low

Calculate Previous Day Return
├── Use Close price (t-1)
└── Use Open price (t-1)
    ├── Subtract Open (t-1) from Close (t-1)
    └── Divide by Open (t-1)

Volume Spike Detection
├── Use Volume (t)
└── Use Volume (t-1)
    ├── Subtract Volume (t-1) from Volume (t)
    └── If Volume Change > 0, Assign 1; Otherwise, Assign 0

Weighted Intraday Return
└── Multiply Intraday Return by Volume Spike

Compute Price Difference Momentum
├── Calculate Today's Close - Yesterday's Close
└── Store in a list for last 5 days

Calculate Average True Range
├── Use High, Low, and Close prices
└── Compute True Range for each day
    ├── Max of (High - Low), (High - Close(t-1)), (Close(t-1) - Low)
    └── Compute the average of the last 5 days' True Ranges

Combine Indicators
├── Add Weighted Intraday Return
├── Add Last 5 Days' Price Difference Momentum
│   └── Sum up the stored values
├── Add Previous Day Return
└── Add Average True Range

Final Alpha Factor
└── Sum of Combined Indicators
```
