│   ├── Compare VWAP with Close price
│   │   ├── If VWAP > Close: Positive Reversal Indicator (1)

│   └── Assign indicator value to Reversal Indicator

│   ├── Multiply Combined Return by Reversal Indicator
│   └── Assign result to Integrated Reversal Return

├── Integrate Volume-Weighted Return with Integrated Reversal Return
│   ├── Add Volume-Weighted Return to Integrated Reversal Return
│   └── Assign result to Initial Alpha Factor
├── Calculate Volatility
│   ├── Compute Daily Returns (Close - Previous Close) / Previous Close
│   ├── Compute 20-day Standard Deviation of Daily Returns
│   └── Assign standard deviation to Volatility
├── Adjust for Volatility
│   ├── Divide Initial Alpha Factor by Volatility
│   └── Assign result to Final Alpha Factor
└── Final Alpha Factor
    └── Assign Final Alpha Factor as the output
