```python
import pandas as pd

def heuristics_v2(df):
    # Calculate Intraday Return
    intraday_range = df['high'] - df['low']
    intraday_return = intraday_range / df['close'].shift(-1)

    # Calculate Overnight Return
    overnight_return = (df['open'] - df['close'].shift(1)) / df['close'].shift(1)

    # Combine Intraday and Overnight Returns
    combined_return = intraday_return + overnight_return

    # Compute Volume Weighted Average Price (VWAP)
    typical_price = (df['high'] + df['low'] + df['close']) / 3
    volume_weighted_price = typical_price * df['volume']
    vwap = volume_weighted_price.sum() / df['volume'].sum()

    # Calculate VWAP Reversal Indicator
    reversal_indicator = (vwap > df['close']).astype(int) - (vwap < df['close']).astype(int)

    # Integrate Reversal Indicator with Combined Return
    final_alpha_factor = combined_return * reversal_indicator

    return final_alpha_factor
```

This function calculates the Intraday Reversal Momentum alpha factor based on the provided thought tree. It first computes the intraday and overnight returns, combines them, then computes the VWAP, and finally integrates the VWAP reversal indicator with the combined return to produce the final alpha factor.
