```txt
Intraday Reversal and Momentum Integration
├── Intraday Return Calculation
│   ├── Compute Intraday Range (High - Low)
│   └── Intraday Return = Intraday Range / Close
├── Overnight Return Calculation
│   ├── Compute Overnight Difference (Open - Previous Close)
│   └── Overnight Return = Overnight Difference / Previous Close
├── Combine Intraday and Overnight Returns
│   ├── Combined Return = Intraday Return + Overnight Return
└── VWAP and Reversal Indicator
    ├── Volume-Weighted Average Price (VWAP) Calculation
    │   ├── Typical Price = (High + Low + Close) / 3
    │   ├── Volume-Weighted Typical Price = Typical Price * Volume
    │   └── VWAP = Sum(Volume-Weighted Typical Price) / Sum(Volume)
    ├── Reversal Indicator
    │   ├── If VWAP > Close: Positive Reversal Indicator (1)
    │   └── If VWAP < Close: Negative Reversal Indicator (-1)
    ├── Integrated Reversal Return
    │   └── Integrated Reversal Return = Combined Return * Reversal Indicator
└── Volume-Weighted Return
    ├── Return Calculation
    │   └── Return = (Close - Open) / Open
    ├── Volume-Weighted Return
    │   └── Volume-Weighted Return = Sum(Return * Volume) / Sum(Volume)
└── Final Alpha Factor
    └── Final Alpha Factor = Integrated Reversal Return + Volume-Weighted Return
```
