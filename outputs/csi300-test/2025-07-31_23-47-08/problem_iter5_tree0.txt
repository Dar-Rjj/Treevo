Intraday Reversal Momentum
├── Calculate Intraday Return
│   ├── Use High price
│   └── Use Low price
│       ├── Compute Intraday Range (High - Low)
│       └── Divide by Close price to get Intraday Return
├── Calculate Overnight Return
│   ├── Use Open price
│   └── Use Previous Close price
│       ├── Compute Difference (Open - Previous Close)
│       └── Divide by Previous Close price to get Overnight Return
├── Combine Intraday and Overnight Returns
│   ├── Add Intraday Return and Overnight Return
│   └── Assign combined return to Combined Return

│   ├── Use Volume
│   ├── Use High, Low, and Close prices
│   │   ├── Compute Typical Price (High + Low + Close) / 3
│   │   └── Multiply Typical Price by Volume
│   └── Sum all volume-weighted typical prices and divide by total volume to get VWAP
├── Calculate VWAP Reversal Indicator
│   ├── Compare VWAP with Close price
│   │   ├── If VWAP > Close: Positive Reversal Indicator (1)
│   │   └── If VWAP < Close: Negative Reversal Indicator (-1)
│   └── Assign indicator value to Reversal Indicator
├── Integrate Reversal Indicator with Combined Return
│   ├── Multiply Combined Return by Reversal Indicator
│   └── Assign result to Integrated Reversal Return
├── Calculate Volume-Weighted Return
│   ├── Use Volume
│   ├── Use Open and Close prices
│   │   ├── Compute Return (Close - Open) / Open
│   │   └── Multiply Return by Volume
│   └── Sum all volume-weighted returns and divide by total volume to get Volume-Weighted Return
├── Integrate Volume-Weighted Return with Integrated Reversal Return
│   ├── Add Volume-Weighted Return to Integrated Reversal Return
│   └── Assign result to Final Alpha Factor
├── Calculate Short-Term Volatility
│   ├── Use High and Low prices
│   │   ├── Compute True Range (max(High - Low, abs(High - Previous Close), abs(Low - Previous Close)))
│   │   └── Sum True Range over a short-term period (e.g., 5 days)
│   └── Divide sum by the number of periods to get Short-Term Volatility
└── Integrate Short-Term Volatility with Final Alpha Factor
    ├── Multiply Final Alpha Factor by Short-Term Volatility
    └── Assign result to Enhanced Alpha Factor
