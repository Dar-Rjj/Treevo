```txt
- Combined Alpha Factor Framework
  - Calculate Intraday and Overnight Returns
    - Intraday Return: (Close_t - Open_t) / Open_t
    - Overnight Return: (Open_t - Close_(t-1)) / Close_(t-1)
  - Aggregate Returns
    - Combined Return: Intraday Return + Overnight Return
  - Momentum Indicators
    - Short-Term Momentum: Close_t - Close_(t-5)
    - Long-Term Momentum: Close_t - Close_(t-20)
    - Momentum Ratio: Short-Term Momentum / Long-Term Momentum
  - Volume Analysis
    - Volume Change: Volume_t - Volume_(t-1)
    - 5-Day EMA of Volume: Exponential Moving Average of the last 5 days' volume
      - EMA_t = (Volume_t * α) + (EMA_(t-1) * (1 - α)), where α = 2 / (n + 1), n=5
    - Volume Adjustment: 
      - If Volume_t > Average(Volume_last_5_days), Multiply Combined Return by 1.5
      - Otherwise, Multiply Combined Return by 0.5
  - Price Volatility
    - Intraday Volatility: High_t - Low_t
    - 20-Day Historical Volatility: Standard Deviation of Daily Returns over the last 20 days
      - Weighted Historical Volatility: 20-Day Historical Volatility * Volume_t
  - Trend Analysis
    - SMA_5: Sum(Close_t, Close_(t-1), ..., Close_(t-4)) / 5
    - SMA_20: Sum(Close_t, Close_(t-1), ..., Close_(t-19)) / 20
    - EMA_12: (Close_t * α) + (EMA_(t-1) * (1 - α)), where α = 2 / (12 + 1)
    - EMA_26: (Close_t * α) + (EMA_(t-1) * (1 - α)), where α = 2 / (26 + 1)
    - MACD Line: EMA_12 - EMA_26
    - Signal Line: 9-period EMA of MACD Line
    - MACD Histogram: MACD Line - Signal Line
  - Relative Strength Index (RSI)
    - RSI_14: 100 - (100 / (1 + (Average Gain / Average Loss)))
      - Where Average Gain = (Sum of gains over 14 periods) / 14
      - Where Average Loss = (Sum of losses over 14 periods) / 14
  - On-Balance Volume (OBV)
    - OBV_t = OBV_(t-1) + (Volume_t if Close_t > Close_(t-1)) - (Volume_t if Close_t < Close_(t-1))
  - Volume-Weighted Average Price (VWAP)
    - VWAP_t = (Cumulative(Typical Price * Volume) / Cumulative(Volume))
      - Typical Price = (High_t + Low_t + Close_t) / 3
  - Order Flow Imbalance
    - Order Flow Ratio: (Buy Volume - Sell Volume) / Total Volume
  - Composite Factor
    - Composite Factor: w1 * (Combined Return * Momentum Ratio * Volume Adjustment) + w2 * (Weighted Historical Volatility) + w3 * (MACD Histogram) + w4 * (RSI_14) + w5 * (OBV_t) + w6 * (Order Flow Ratio)
  - Generate Alpha Signals
    - Alpha Signal: if (Composite Factor > threshold) then buy, else sell
  - Final Alpha Factor
    - Integrate All Components
    - Output the Final Alpha Factor
```
