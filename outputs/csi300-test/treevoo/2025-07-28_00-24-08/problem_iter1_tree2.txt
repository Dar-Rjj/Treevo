Trend Reversal with Volume Confirmation
├── Calculate Daily Price Change
│   ├── Obtain Close price
│   └── Subtract Previous Day's Close price

│   ├── If Daily Price Change > 0

│   └── If Daily Price Change <= 0

│   ├── Obtain Current Day's Volume
│   └── Subtract Previous Day's Volume
├── Assess Volume Confirmation
│   ├── If Volume Delta > 0 and Daily Price Change > 0
│   │   └── Confirm as Strong Uptrend
│   └── If Volume Delta < 0 and Daily Price Change < 0
│       └── Confirm as Strong Downtrend
└── Factor Calculation
    ├── If Trend is Uptrend and Volume Confirmation is Strong Uptrend
    │   ├── Multiply by Positive Weight (e.g., 1.5)
    │   └── Output as Factor Value
    └── If Trend is Downtrend and Volume Confirmation is Strong Downtrend
        ├── Multiply by Negative Weight (e.g., -1.5)
        └── Output as Factor Value
