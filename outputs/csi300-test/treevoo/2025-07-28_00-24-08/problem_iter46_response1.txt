Please note that the condition `if df['volume'] > avg_volume` is not a valid way to apply a conditional operation in a vectorized manner for a Pandas DataFrame. Instead, we should use `np.where` or `pd.Series.where` to apply the condition. Here is the corrected version:

```python
import pandas as pd
import numpy as np

def heuristics_v2(df):
    # Calculate Intraday Return
    intraday_return = (df['close'] - df['open']) / df['open']

    # Calculate Previous Day's Overnight Return
    overnight_return = (df['open'] - df['close'].shift(1)) / df['close'].shift(1)

    # Combine Returns
    combined_returns = intraday_return * overnight_return

    # Short-Term Price Momentum
    short_term_momentum = df['close'] - df['close'].shift(5)

    # Long-Term Price Momentum
    long_term_momentum = df['close'] - df['close'].shift(20)

    # Price Momentum Ratio
    price_momentum_ratio = short_term_momentum / long_term_momentum

    # Volume Change Indicator
    volume_change = df['volume'] - df['volume'].shift(5)

    # Integrate Factors
    factor = combined_returns * price_momentum_ratio
    avg_volume = df['volume'].mean()
    factor = factor * np.where(df['volume'] > avg_volume, 1.7, 0.3)

    return factor
```
