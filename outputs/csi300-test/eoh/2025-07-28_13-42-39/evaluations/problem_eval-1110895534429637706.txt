import pandas as pd

def heuristics_v2(df):
    def apply_heuristic(row):
        if (row['high'] - row['low']) / row['close'] > 0.05 and row['volume'] > df['volume'].mean():
            return 1
        elif (row['high'] - row['low']) / row['close'] < 0.02 and row['volume'] < df['volume'].mean():
            return -1
        else:
            return 0

    heuristics_matrix = df.apply(apply_heuristic, axis=1)
    return heuristics_matrix
```

This Python function `heuristics_v2` generates a Series of heuristic factor values based on the input DataFrame's market features, without normalizing the data, reflecting potential signals for future stock return heuristics_matrix
