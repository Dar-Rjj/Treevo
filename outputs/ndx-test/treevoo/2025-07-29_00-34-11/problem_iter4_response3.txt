```python
def heuristics_v2(data):
    # Calculate High-Low Spread
    high_low_spread = data['high'] - data['low']
    
    # Calculate Price Momentum
    price_momentum = (data['close'] - data['close'].shift(1)) / data['close'].shift(1)
    
    # Multiply High-Low Spread with Price Momentum
    spread_momentum = high_low_spread * price_momentum
    
    # Divide the Result by Volume
    factor_values = spread_momentum / data['volume']
    
    return factor_values
```
