```python
def heuristics_v2(df):
    # Calculate Intraday High-Low Difference
    intraday_diff = df['high'] - df['low']
    
    # Calculate Volume-Amount Ratio
    volume_amount_ratio = df['volume'] / df['amount']
    
    # Incorporate Volume and Amount for Weighting
    weighted_intraday_diff = intraday_diff * volume_amount_ratio
    
    return weighted_intraday_diff
```
