```python
def heuristics_v2(df):
    # Calculate Intraday Range
    intraday_range = df['high'] - df['low']
    
    # Calculate Previous Day's Return
    previous_day_return = (df['close'] / df['open'].shift(1)) - 1
    
    # Inverse of Intraday Range
    inv_intraday_range = 1 / intraday_range
    
    # Momentum Adjustment
    adjusted_momentum = previous_day_return * inv_intraday_range
    
    return adjusted_momentum
```
