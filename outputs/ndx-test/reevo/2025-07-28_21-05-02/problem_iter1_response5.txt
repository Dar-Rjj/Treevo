```python
def heuristics_v2(df: pd.DataFrame) -> pd.Series:
    # Calculate momentum factor using close price over the last 5 days
    df['momentum'] = df['close'].pct_change(periods=5)
    # Calculate the average volume over the last 10 days
    df['avg_volume'] = df['volume'].rolling(window=10).mean()
    # Combine the momentum with a scaled version of the difference between current volume and average volume
    # This captures the idea of high momentum with unusual volume
    factor = df['momentum'] + (df['volume'] - df['avg_volume']) / df['avg_volume']
    return factor
```
