```python
def heuristics_v2(df: pd.DataFrame) -> pd.Series:
    # Calculate the difference between today's close and the previous day's close
    price_change = df['close'].diff()
    # Calculate the volume weighted by the absolute value of the price change
    volume_weighted_change = df['volume'] * abs(price_change)
    # The factor is a smoothed (using 3-day rolling mean) version of this volume weighted change
    smoothed_volume_weighted_change = volume_weighted_change.rolling(window=3).mean()
    return smoothed_volume_weighted_change
```
