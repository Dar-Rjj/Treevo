Alpha Mining Thought Tree
- Calculate Momentum Factors
  - Short-Term Momentum: (Close_t / Close_(t-1)) - 1
  - Medium-Term Momentum: (Close_t / Close_(t-20)) - 1
  - Long-Term Momentum: (Close_t / Close_(t-252)) - 1
- Compute Volatility Indicators
  - Price Range: High_t - Low_t
  - Average True Range (ATR) over 14 days
  - Historical Volatility over 20 days
- Identify Volume Trends
  - Volume Change: Volume_t - Volume_(t-1)
  - Relative Volume: Volume_t / Average(Volume_past_30_days)
- Analyze Market Sentiment Using Prices
  - Bullish/Bearish Candles: If (Close_t > Open_t) then 1 else 0
  - Reversal Signals: If (Close_t < Open_t AND Close_(t-1) > Open_(t-1)) then -1, if (Close_t > Open_t AND Close_(t-1) < Open_(t-1)) then 1, else 0
- Generate Technical Indicators
  - Moving Averages
    - Simple Moving Average (SMA) over 50 and 200 days
    - SMA Crosses: If (SMA_50_t > SMA_200_t AND SMA_50_(t-1) < SMA_200_(t-1)) then 1, if (SMA_50_t < SMA_200_t AND SMA_50_(t-1) > SMA_200_(t-1)) then -1, else 0
  - Exponential Moving Averages (EMA)
    - EMA over 12 and 26 days
    - MACD Line: EMA_12_t - EMA_26_t
    - Signal Line: EMA of MACD Line over 9 days
  - Bollinger Bands
    - Middle Band: 20-day SMA
    - Upper Band: Middle_Band + 2 * Standard_Deviation(Close, 20)
    - Lower Band: Middle_Band - 2 * Standard_Deviation(Close, 20)
    - Bollinger Bandwidth: (Upper_Band - Lower_Band) / Middle_Band
- Evaluate Liquidity
  - Dollar Volume: Amount_t
  - Turnover Ratio: Amount_t / (Average(Amount_past_30_days))
- Assess Value
  - Price-to-Average Ratio: Close_t / Average(Close_past_20_days)
- Pattern Recognition
  - Candlestick Patterns
    - Doji: If (Abs(Close_t - Open_t) <= 0.1 * (High_t - Low_t)) then 1 else 0
    - Hammer/Inverted Hammer: Specific conditions based on candle structure
- Combine Factors
  - Create composite score by assigning weights to each factor based on perceived predictive power
