```txt
Alpha Factor Thought Tree
- Trend Analysis
  - Moving Averages
    - Simple Moving Average (SMA)
      - Calculate SMA for close prices over a short period (e.g., 5 days)
      - Calculate SMA for close prices over a long period (e.g., 20 days)
    - Exponential Moving Average (EMA)
      - Calculate EMA for close prices over a short period (e.g., 5 days)
      - Calculate EMA for close prices over a long period (e.g., 20 days)
  - Momentum Indicators
    - Price Rate of Change (ROC)
      - Calculate ROC for close prices over a period (e.g., 12 days)
    - Relative Strength Index (RSI)
      - Calculate RSI for close prices with a period (e.g., 14 days)
- Volume Analysis
  - On-Balance Volume (OBV)
    - Calculate OBV using close price and volume
  - Volume Trend
    - Calculate the difference between current day's volume and the previous day's volume
- Volatility Analysis
  - True Range
    - Calculate the true range using high, low, and close prices
  - Average True Range (ATR)
    - Calculate ATR using true range over a period (e.g., 14 days)
- Price Pattern Analysis
  - Inside Bar
    - Identify inside bar patterns: High(t) < High(t-1) and Low(t) > Low(t-1)
  - Outside Bar
    - Identify outside bar patterns: High(t) > High(t-1) and Low(t) < Low(t-1)
- Order Flow Analysis
  - Accumulation/Distribution Line (A/D)
    - Calculate A/D line using high, low, and close prices along with volume
  - Money Flow Index (MFI)
    - Calculate MFI using high, low, close, and volume over a period (e.g., 14 days)
- Sentiment Analysis
  - Bullish/Bearish Candlestick Patterns
    - Identify bullish and bearish candlestick patterns using open, high, low, and close prices
- Market Strength Analysis
  - Stochastic Oscillator
    - Calculate %K and %D lines using high, low, and close prices over a period (e.g., 14 days)
  - Williams %R
    - Calculate Williams %R using high, low, and close prices over a period (e.g., 14 days)
- Composite Factors
  - Combine multiple factors to create a composite score
    - Example: (SMA_short - SMA_long) + (RSI - 50) + (OBV_change)
```
