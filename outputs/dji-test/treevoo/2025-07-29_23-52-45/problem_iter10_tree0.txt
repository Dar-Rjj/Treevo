```txt
Volume-Weighted and Volatility Adjusted Hybrid Momentum-Reversal Signal
├── Calculate Intraday Volatility
│   └── intraday_volatility = |High - Low|
├── Calculate Price Momentum
│   ├── Define lookback period (e.g., 20 days)
│   ├── SMA(Close, lookback_period)
│   └── Momentum_signal = Close - SMA(Close, lookback_period)
├── Calculate Volume Averaged Momentum
│   ├── Define lookback period for volume (e.g., 20 days)
│   ├── AVG(Volume, lookback_period)
│   └── Volume_adjusted_momentum = Momentum_signal * Volume / AVG(Volume, lookback_period)
├── Adjust for Close Price Volatility
│   ├── Define lookback period (e.g., 20 days)
│   ├── STD(Close, lookback_period)
│   └── Final_volume_weighted_momentum = Volume_adjusted_momentum / STD(Close, lookback_period)
├── Calculate Short-Term Return
│   └── short_term_return = (Close[t] - Close[t-5]) / Close[t-5]
├── Calculate Long-Term Return
│   └── long_term_return = (Close[t] - Close[t-60]) / Close[t-60]
├── Combine Momentum Factors with Short-Term Return
│   └── combined_momentum_factor = 0.7 * Final_volume_weighted_momentum + 0.3 * Volume[t] * short_term_return
├── Subtract Long-Term Return Component
│   └── long_term_return_weighted = long_term_return * Volume[t]
│   └── hybrid_momentum_reversal_signal = combined_momentum_factor - long_term_return_weighted
└── Reversal Indicator
    └── reversal_indicator = -1 * hybrid_momentum_reversal_signal
```
