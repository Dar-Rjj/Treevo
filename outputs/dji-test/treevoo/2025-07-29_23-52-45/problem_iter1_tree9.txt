Relative Strength Index with Volume Adjustment (RSIVA)
├── Calculate Daily Price Change
│   ├── Use Close price
│   └── Subtract Previous Close price
├── Separate Positive and Negative Changes
│   ├── If change > 0, Assign to Positive Changes
│   └── If change < 0, Assign to Negative Changes
├── Calculate Average Positive and Negative Changes over 14 Days
│   ├── Average Positive Changes
│   │   ├── Sum Positive Changes
│   │   └── Divide by 14
│   └── Average Negative Changes
│       ├── Sum Negative Changes
│       └── Divide by 14
├── Compute RSIVA
│   ├── Calculate Relative Strength (RS)
│   │   ├── Use Average Positive Changes
│   │   └── Divide by Absolute of Average Negative Changes
│   └── Calculate RSI
│       ├── Apply RSI Formula: 100 - (100 / (1 + RS))
├── Adjust by Volume
│   ├── Calculate Volume Change
│   │   ├── Use Current Volume
│   │   └── Subtract Previous Volume
│   ├── Normalize Volume Change
│   │   ├── If Volume Change > 0, Assign 1
│   │   └── If Volume Change < 0, Assign -1
│   └── Multiply RSI by Normalized Volume Change
└── Output RSIVA
