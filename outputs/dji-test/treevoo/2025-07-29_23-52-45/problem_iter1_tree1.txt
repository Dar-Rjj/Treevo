```txt
High-Low Range Momentum
├── Calculate Daily High-Low Range
│   ├── Use High price
│   └── Use Low price
├── Compute Moving Average of High-Low Range
│   ├── Apply Simple Moving Average (SMA)
│   │   └── Set lookback period to 10 days
│   └── Apply Exponential Moving Average (EMA)
│       └── Set smoothing factor to 0.2
└── Determine Momentum
    ├── Compare Current High-Low Range with SMA
    ├── Compare Current High-Low Range with EMA
    └── Assign Signal
        ├── If Current > SMA and Current > EMA: Strong Buy
        ├── If Current > SMA but Current < EMA: Weak Buy
        ├── If Current < SMA but Current > EMA: Weak Sell
        └── If Current < SMA and Current < EMA: Strong Sell
```
