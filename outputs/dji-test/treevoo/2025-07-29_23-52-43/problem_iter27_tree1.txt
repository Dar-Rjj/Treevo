Intraday, Breakout, and Volume Momentum Composite
├── Calculate Intraday Return
│   └── (High - Low) / Low
├── Calculate Close-to-PreviousClose Return
│   └── (Close - PreviousClose) / PreviousClose
├── Calculate High-Low Breakout Intensity
│   ├── (High - Low) / Close
│   └── If Breakout Intensity < 0, set to 0
├── Volume Weighted Intraday Return
│   ├── (Close_t - Open_t) / Open_t
│   └── Multiply by (Volume_t / Avg(Volume_t-5...t-1))
├── Volume Spike Detection
│   ├── Daily Volume Change
│   │   ├── Current Day Volume (Volume_t)
│   │   └── Previous Day Volume (Volume_t-1)
│   ├── Spike Threshold
│   │   └── 90th Percentile (Past 21 Days' Volume Changes) * Multiplier (2.0)
│   └── Apply Spike Condition
├── Volume Increase Ratio
│   └── (Volume_t / Volume_t-1)
├── Price Volatility
│   └── Compute (StdDev(Close_t-5...t) / Close_t-1)
├── Intermediate Factor
│   └── ((Volume_t / Volume_t-1) * (Close_t - Avg(Close_t-5...t-1)) * (StdDev(Close_t-5...t) / Close_t-1))
├── Adjust for Volatility
│   └── Divide Combined Metrics by Rolling Volatility
│       └── Standard Deviation of (High - Low) over a Window
├── Trend Following Indicator
│   └── (Close_t - Close_t-1) / StdDev(Close_t-5...t-1)
├── Volume Momentum Indicator
│   └── (Volume_t - Avg(Volume_t-5...t-1)) / StdDev(Volume_t-5...t-1)
└── Final Composite Factor
    ├── Volume Weighted Intraday Return
    ├── Intermediate Factor
    ├── Adjusted Combined Metrics
    ├── Trend Following Indicator
    └── Volume Momentum Indicator
