```txt
Price-Volume-Adjusted Momentum with Price Gap and Range Integration
├── Calculate Adjusted Daily Return
│   ├── Calculate Daily Return using (Close_t - Close_{t-1})
│   ├── Calculate Price Gap using (Open_t - Close_{t-1})
│   └── Adjust Daily Return: If (Gap > 0) Subtract Gap; If (Gap < 0) Add Gap
├── Calculate Price Range Momentum
│   ├── Current Period High-to-Low Range (High_t - Low_t)
│   ├── Previous Period High-to-Low Range (High_{t-1} - Low_{t-1})
│   └── Subtract Previous from Current Range
├── Volume Change Factor (Volume_t / Volume_{t-1})
└── Combine Factors
    ├── Multiply Adjusted Daily Return by Price Range Momentum
    └── Multiply the Result by Volume Change
```
