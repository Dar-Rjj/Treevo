```python
def heuristics_v2(df):
    # Calculate High-Low Spread
    high_low_spread = df['high'] - df['low']
    
    # Adjust by Volume and Close-Open Difference
    previous_volume = df['volume'].shift(1)
    close_open_return = (df['close'] - df['open']) / df['open']
    
    # Calculate the High-to-Low Volatility Weighted Alpha
    alpha = high_low_spread * previous_volume * close_open_return
    
    return alpha
```
