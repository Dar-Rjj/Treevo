```python
def heuristics_v2(df):
    # Calculate the ratio of close to open price, which can be a simple indicator of the day's performance
    close_open_ratio = df['close'] / df['open']
    
    # Calculate the ratio of high to low price, which gives an idea of the day's volatility
    high_low_ratio = df['high'] / df['low']
    
    # Calculate the ratio of amount to volume, which can be a proxy for average transaction size
    avg_transaction_size = df['amount'] / df['volume']
    
    # Combine the three ratios in a weighted sum to form the alpha factor
    # The weights are chosen such that a higher value suggests a positive outlook
    alpha_factor = 0.5 * close_open_ratio + 0.3 * high_low_ratio + 0.2 * avg_transaction_size
    
    return alpha_factor
```
