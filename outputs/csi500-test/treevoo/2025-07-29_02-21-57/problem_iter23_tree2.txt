Explore Price Patterns, Volume Indicators, and Momentum

├── Calculate Moving Averages
│   ├── Simple Moving Average (SMA) of Close Prices
│   │   ├── SMA_10 = Sum(Close[i] for i in range(t-9, t+1)) / 10
│   │   └── SMA_20 = Sum(Close[i] for i in range(t-19, t+1)) / 20
│   └── Exponential Moving Average (EMA) of Close Prices
│       ├── EMA_10 = (Close[t] * (2 / (1 + 10))) + (EMA_10[t-1] * (1 - (2 / (1 + 10))))
│       └── EMA_20 = (Close[t] * (2 / (1 + 20))) + (EMA_20[t-1] * (1 - (2 / (1 + 20))))

├── Analyze Volatility
│   ├── Calculate True Range
│   │   ├── TR1: High[t] - Low[t]
│   │   ├── TR2: Abs(High[t] - Close[t-1])
│   │   ├── TR3: Abs(Low[t] - Close[t-1])
│   │   └── True Range: Max(TR1, TR2, TR3)
│   └── Calculate Average True Range (ATR)
│       └── ATR_14 = (Sum(TR[i] for i in range(t-13, t+1)) / 14)

├── Identify Breakouts
│   ├── New Highs or Lows
│   │   ├── If High[t] > Max(High[i] for i in range(t-5, t)), then New_High = True
│   │   └── If Low[t] < Min(Low[i] for i in range(t-5, t)), then New_Low = True
│   └── Measure Momentum
│       ├── Rate of Change (ROC) of Close Prices
│       │   └── ROC_12 = (Close[t] - Close[t-12]) / Close[t-12]
│       └── Relative Strength Index (RSI)
│           └── RSI_14 = 100 - (100 / (1 + (Average Gain / Average Loss)))
│               Where:
│                   ├── Average Gain = Sum(Max(Close[i] - Close[i-1], 0) for i in range(t-13, t+1)) / 14
│                   └── Average Loss = Abs(Sum(Min(Close[i] - Close[i-1], 0) for i in range(t-13, t+1)) / 14)

├── Assess Volume Changes
│   ├── On-Balance Volume (OBV)
│   │   └── OBV[t] = OBV[t-1] + (Volume[t] if Close[t] > Close[t-1] else (-Volume[t] if Close[t] < Close[t-1] else 0))
│   └── Examine Price and Volume Relationships
│       ├── Money Flow Index (MFI)
│       │   └── MFI_14 = 100 - (100 / (1 + (Positive Money Flow / Negative Money Flow)))
│       │       Where:
│       │           ├── Positive Money Flow = Sum(Raw Money Flow[i] if Raw Money Flow[i] > 0 for i in range(t-13, t+1))
│       │           ├── Negative Money Flow = Abs(Sum(Raw Money Flow[i] if Raw Money Flow[i] < 0 for i in range(t-13, t+1)))
│       │           └── Raw Money Flow = (High[i] + Low[i] + Close[i]) / 3 * Volume[i]
│       └── Accumulation/Distribution Line (ADL)
│           └── ADL[t] = ADL[t-1] + ((Close[t] - Low[t]) - (High[t] - Close[t])) / (High[t] - Low[t]) * Volume[t] if High[t] != Low[t] else 0

├── Calculate Intraday High-Low Spread
│   └── Subtract Low from High

├── Calculate Close-to-Open Return
│   └── Subtract Open from Close

├── Calculate Price Momentum
│   └── Subtract Close price 14 days ago from Current Close
