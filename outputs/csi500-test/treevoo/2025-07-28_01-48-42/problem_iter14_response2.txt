```python
import pandas as pd

def heuristics_v2(df):
    """
    This function computes a novel alpha factor based on the given thought tree.
    The alpha factor is computed as the weighted sum of the percentage change in close price and the ratio of volume to amount.
    
    Parameters:
    df (pd.DataFrame): A pandas DataFrame with columns ['open', 'high', 'low', 'close', 'amount', 'volume'] and index as date.
    
    Returns:
    pd.Series: A pandas Series indexed by date representing the factor values.
    """
    # Calculate the percentage change in close price
    close_pct_change = df['close'].pct_change()
    
    # Calculate the ratio of volume to amount
    vol_to_amount_ratio = df['volume'] / df['amount']
    
    # Compute the weighted sum of the two components
    alpha_factor = 0.7 * close_pct_change + 0.3 * vol_to_amount_ratio
    
    return alpha_factor
```
