│   │       └── True Range = High - Previous Close if High > Previous Close
│   │           └── Or,
│   │               └── True Range = Previous Close - Low if Low < Previous Close
└── Calculate Moving Average of True Range
    ├── Compute the average of True Range over a given period
    └── Use the result as momentum indicator
