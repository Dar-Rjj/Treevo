```txt
Alpha Factor Generation
- Momentum Indicators
  - Simple Moving Average (SMA)
    - Calculate the 50-day and 200-day SMAs
      - SMA_50 = (Close[t] + Close[t-1] + ... + Close[t-49]) / 50
      - SMA_200 = (Close[t] + Close[t-1] + ... + Close[t-199]) / 200
    - Calculate the difference between 50-day and 200-day SMAs
      - SMA_Diff = SMA_50 - SMA_200
- Volume Indicators
  - On-Balance Volume (OBV)
    - Initialize OBV at day 0 as 0
    - For each subsequent day, update OBV
      - If Close[t] > Close[t-1], OBV[t] = OBV[t-1] + Volume[t]
      - If Close[t] < Close[t-1], OBV[t] = OBV[t-1] - Volume[t]
      - If Close[t] == Close[t-1], OBV[t] = OBV[t-1]
- Price Pattern Indicators
  - Doji Candlestick Pattern
    - Identify if the open and close are nearly equal
      - IsDoji = abs(Open[t] - Close[t]) / (High[t] - Low[t]) < 0.01
- Volatility Indicators
  - True Range (TR)
    - Calculate the true range for each day
      - TR[t] = max(High[t] - Low[t], abs(High[t] - Close[t-1]), abs(Low[t] - Close[t-1]))
  - Average True Range (ATR) over 14 days
    - ATR = (TR[t] + TR[t-1] + ... + TR[t-13]) / 14
- Market Sentiment Indicators
  - Money Flow Index (MFI)
    - Calculate typical price and raw money flow
      - TypicalPrice = (High[t] + Low[t] + Close[t]) / 3
      - RawMoneyFlow = TypicalPrice * Volume[t]
    - Determine positive and negative money flow
      - PositiveMF = sum(RawMoneyFlow[t] if TypicalPrice[t] > TypicalPrice[t-1])
      - NegativeMF = sum(RawMoneyFlow[t] if TypicalPrice[t] < TypicalPrice[t-1])
    - Calculate MFI
      - MFI = 100 - (100 / (1 + (PositiveMF / NegativeMF)))
```
